## 一、搭搭 (DADA) - AI智能电子衣橱

### 1.1 产品概述

搭搭是一款以衣橱管理为核心的AI穿搭推荐工具，由深圳衫叠科技有限公司开发。产品定位为"200万用户共建的AI智能电子衣橱APP"，主打衣橱数字化管理+智能搭配推荐。

### 1.2 核心功能

| 功能 | 描述 |
|------|------|
| 电子衣橱 | 拍照上传衣物，AI自动识别类别/颜色/属性，数字化管理个人衣橱 |
| AI搭配推荐 | 基于已有单品，AI生成搭配方案，支持DIY搭配和AI推荐两种模式 |
| 今日穿搭 | 根据天气温度智能推荐每日OOTD，支持异地规划（旅游/出差） |
| 穿搭日记 | 记录每日穿搭，形成个人穿搭档案 |
| 统计分析 | 衣橱使用率分析、消费统计、穿搭频率统计 |

### 1.3 商业模式

搭搭采用**免费+增值服务**模式。基础衣橱管理和AI搭配推荐免费，高级功能（如无限衣橱容量、高级搭配模板、穿搭数据分析）通过内购解锁。推广策略以精准素人短视频为主，每条视频制作成本仅30-200元，依靠偶发爆款带来转化（来源：智源社区对话搭搭团队）。

### 1.4 优势

1. **先发优势明显**：在AI衣橱管理赛道中用户量领先（200万用户），用户迁移成本高（已上传大量衣物照片）
2. **需求切入精准**：从"衣橱管理"这个高频刚需切入，用户留存天然高
3. **低成本获客**：素人短视频推广模式成本低、信任度高，符合产品调性
4. **AI识别能力**：自动识别衣物类别/颜色/属性，降低用户录入成本

### 1.5 弱点

1. **推荐质量有限**：AI搭配仅基于已有单品做排列组合，缺乏审美理解和场景适配，用户反馈"推荐搭配不好看"
2. **无对话交互**：不支持自然语言对话，用户无法描述需求（如"明天面试穿什么"），只能被动接受推荐
3. **无可视化**：搭配结果以单品图片平铺展示，无穿搭效果图或虚拟试穿，用户需要脑补上身效果
4. **无电商闭环**：推荐的都是用户已有单品，无法推荐新商品，无购买转化路径
5. **社区缺失**：纯工具型产品，无社区互动，用户粘性依赖功能使用频率

### 1.6 用户量级

约200万注册用户（来源：App Store描述及多个安卓应用商店），App Store评分5.0但仅2个评分，说明活跃用户规模有限。

### 1.7 对AiNeed的启示

搭搭验证了"AI衣橱管理+搭配推荐"的需求存在，但其产品形态停留在"工具"层面。AiNeed需要从"工具"升级为"造型师"——通过对话式AI提供个性化建议，通过文生图提供可视化，通过CPS导购和POD定制提供购买闭环。

## 二、小红书 - 生活方式社区+穿搭分享

### 2.1 产品概述

小红书是中国最大的生活方式社区平台，以UGC内容为核心、搜索为纽带、电商为闭环。穿搭/时尚是小红书最大的内容品类之一，每日新增40万条女装鞋靴相关笔记。

### 2.2 核心功能

| 功能 | 描述 |
|------|------|
| 笔记发布 | 图文/视频笔记，支持标签/话题/位置 |
| 双列Feed流 | 瀑布流内容推荐，算法驱动个性化分发 |
| 搜索 | 生活方式搜索引擎，穿搭/美妆/旅行等垂直搜索 |
| 电商闭环 | 商城+直播带货+品牌合作，"种收一体" |
| 社区互动 | 关注/点赞/收藏/评论/私信 |

### 2.3 商业模式

小红书的商业模式为**广告+电商双轮驱动**。广告收入来自品牌合作（蒲公英平台）、信息流广告和搜索广告；电商收入来自自营商城和直播带货佣金。2025年平台MAU突破3.5亿，DAU突破1亿，用户日均使用时长超70分钟（来源：QuestMobile统计及小红书官方数据）。含购物车链接的测评类笔记GMV转化率比普通笔记高4倍，用户客单价突破500元（来源：未来智库）。

### 2.4 优势

1. **用户规模巨大**：3.5亿MAU，1亿DAU，是中国最大的生活方式社区
2. **内容生态成熟**：每日40万条女装笔记+145万条评论，穿搭内容供给充足
3. **种草心智强**：用户来小红书的第一动机就是"找灵感"，消费决策链路短
4. **"种收一体"闭环**：从内容种草到电商转化全链路贯通
5. **用户画像优质**：95后占比50%、00后占比35%，一二线城市用户达50%，高净值年轻女性为主

### 2.5 弱点

1. **搜索效率低**：用户需要自己搜索+筛选穿搭灵感，无法个性化推荐"适合我"的搭配
2. **内容同质化**：68%的创作者面临内容同质化困境（来源：原创力文档），穿搭笔记大量重复
3. **推荐不精准**：算法推荐基于内容相似度，不考虑用户体型/肤色/风格等个人特征
4. **无法交互**：用户无法与内容对话（如"这套搭配适合梨形身材吗"），只能被动浏览
5. **头部集中**：头部账号流量占比达45%，中小博主增长困难

### 2.6 用户量级

3.5亿MAU（2025年底），1亿DAU，日均使用时长70分钟。

### 2.7 对AiNeed的启示

小红书是AiNeed最大的"竞品"也是最大的"合作伙伴"。AiNeed不应试图复制小红书的社区，而应定位为"小红书的AI增强层"——用户在小红书找灵感，在AiNeed获得个性化推荐和可视化体验。社区功能参考小红书的双列Feed流和种草模式，但以AI推荐为核心差异化。

## 三、得物 (Dewu) - 潮流电商

### 3.1 产品概述

得物是中国第四大电商平台（仅次于淘宝、拼多多、京东），以"先鉴别后发货"模式建立正品信任，主打潮流运动鞋服。累计用户超过5亿，30岁以下年轻人渗透率达50%，90后占比超90%，95后渗透率高达70%（来源：人民网/得物商家大会）。

### 3.2 核心功能

| 功能 | 描述 |
|------|------|
| 正品鉴别 | "先鉴别后发货"，专业团队验证真伪后发货 |
| 潮流社区 | 用户发布穿搭/开箱/测评内容 |
| 限量发售 | 独家限量款发售，制造稀缺性 |
| 品牌商城 | 品牌官方入驻，正品保障 |
| 3D试穿 | 部分鞋款支持AR试穿 |

### 3.3 商业模式

得物的商业模式为**电商交易佣金+广告**。平台采用"出价即经营"半托管模式，商家仅需选品、上架与发货，无需组建客服/设计/投流团队。佣金率根据品类不同，通常在5-15%。2024年3月MAU为8,800万，同比增长17.3%（来源：Daxue Consulting）。

### 3.4 优势

1. **正品信任壁垒**："先鉴别后发货"模式在潮鞋/潮服领域建立了极强的信任壁垒
2. **年轻用户浓度极高**：90后占比超90%，95后渗透率70%，是中国年轻人浓度最高的电商平台
3. **品牌合作深度**：与Nike/Adidas/李宁等头部品牌深度合作，独家发售权
4. **社区+电商闭环**：内容种草→交易转化链路完整
5. **品类扩展成功**：从潮鞋扩展到运动户外、美妆个护、家居、黄金珠宝等

### 3.5 弱点

1. **无AI造型推荐**：纯电商模式，无个性化搭配推荐能力
2. **价格偏高**：正品鉴别模式导致价格高于其他平台，不适合预算有限的用户
3. **女性用户占比低**：虽然近年女性用户占比提升，但仍以男性潮鞋用户为主
4. **无定制服务**：标准化商品为主，不支持个性化定制
5. **社区内容偏展示**：社区以开箱/测评为主，缺乏搭配教学和灵感启发

### 3.6 用户量级

累计用户超5亿，2024年3月MAU 8,800万，平台用户规模超6亿（来源：得物2025年度好生意榜）。

### 3.7 对AiNeed的启示

得物验证了"社区+电商"模式在时尚领域的可行性，以及年轻人对正品信任的需求。AiNeed的CPS导购可以对接得物等平台，但不应复制其电商模式。AiNeed的差异化在于"AI推荐+定制"，而非"正品鉴别+交易"。

## 四、ZEPETO - 虚拟形象社交

### 4.1 产品概述

ZEPETO是韩国Naver Z旗下的3D虚拟形象社交平台，用户上传自拍即可生成3D虚拟形象，在虚拟世界中社交、换装、创作。全球注册用户超4亿，月活用户约2,000万，95%来自韩国以外市场。ZEPETO Studio已向虚拟时尚设计师支付超1.2亿美元（来源：Customuse/Asia Bits）。

### 4.2 核心功能

| 功能 | 描述 |
|------|------|
| 3D虚拟形象 | 上传自拍生成3D头像，高度自定义面部/发型/服装 |
| 虚拟世界 | 3D虚拟空间，用户可以社交、探索、玩游戏 |
| 虚拟时装 | 购买/设计虚拟服装，已售出超25亿件虚拟物品 |
| 创作者经济 | 设计师可创作并出售虚拟服装，获得分成 |
| 品牌合作 | Gucci/Dior/Nike/Versace/Coach等品牌限定虚拟服装 |

### 4.3 商业模式

ZEPETO的商业模式为**虚拟物品销售+品牌合作+创作者分成**。用户购买虚拟服装和道具（使用平台虚拟货币ZEM），创作者获得销售分成。品牌方通过限定虚拟服装发售获得品牌曝光和收入。ZEPETO是独角兽企业，估值超10亿美元（来源：Asia Bits）。

### 4.4 优势

1. **虚拟形象技术领先**：3D形象生成质量高，用户自定义程度深
2. **创作者生态成熟**：设计师可以创作并销售虚拟服装，形成UGC生态
3. **品牌合作资源丰富**：Gucci/Dior/Nike等顶级品牌入驻
4. **全球化用户基础**：4亿注册用户，95%来自海外，跨文化吸引力强
5. **虚拟物品经济**：已售出25亿件虚拟物品，验证了虚拟时装的付费意愿

### 4.5 弱点

1. **偏娱乐社交**：核心体验是虚拟世界社交，与真实时尚穿搭关联弱
2. **无真实服装推荐**：虚拟换装无法映射到真实购买决策
3. **用户年龄偏低**：核心用户为13-18岁青少年，消费力有限
4. **3D形象制作成本高**：高质量3D服装需要专业建模，创作者门槛高
5. **无AI造型师**：无个性化搭配推荐，用户需要自己搭配虚拟服装

### 4.6 用户量级

4亿+注册用户，2,000万MAU，已售出25亿+虚拟物品。

### 4.7 对AiNeed的启示

ZEPETO验证了虚拟形象的付费意愿和情绪价值，但它的3D路线对创业团队成本过高。AiNeed的Q版形象采用react-native-skia动态绘制，成本极低（零外包/零GPU/零CDN），但需要保证审美质量。关键差异化：AiNeed的Q版形象不是社交玩具，而是穿搭可视化工具——换装结果映射到真实搭配推荐。

## 五、Stitch Fix - 美国AI造型订阅

### 5.1 产品概述

Stitch Fix是美国AI驱动的个人造型订阅服务平台，用户填写风格问卷后，AI+人类造型师为其挑选5-8件服装寄送到家，用户留下喜欢的、退回不喜欢的。2025财年活跃客户约231万，人均年消费549美元（来源：Stitch Fix FY2025财报/SEC文件）。

### 5.2 核心功能

| 功能 | 描述 |
|------|------|
| 风格问卷 | 详细的生活方式/体型/预算/风格偏好问卷 |
| AI+人工造型 | AI算法初选+人类造型师微调，5-8件服装寄送 |
| Fix订阅 | 定期寄送搭配盒子，用户保留喜欢的退回不喜欢的 |
| Freestyle | 用户可直接购买AI推荐的单品，无需订阅盒子 |
| Stitch Fix Vision | 2025年推出的生成式AI造型可视化平台，用户可看到自己穿上推荐服装的效果 |

### 5.3 商业模式

Stitch Fix的商业模式为**服装销售+订阅服务**。收入来自用户购买的服装（保留的Fix盒子中的单品+Freestyle直接购买），不收取造型服务费。2025财年每位活跃客户贡献收入549美元，同比增长3%。公司2025财年实现盈利转折，AI工具Stitch Fix Vision使75%用户重复使用，Freestyle消费在90天内提升100%以上（来源：Market Minute/Zacks）。

### 5.4 优势

1. **AI造型验证成功**：AI+人工混合模式在成熟市场验证了"AI推荐+人工微调"的可行性
2. **高客单价**：人均年消费549美元，用户将Stitch Fix视为主要衣橱解决方案
3. **数据飞轮**：10年+用户行为数据积累，推荐算法持续优化
4. **Vision平台突破**：生成式AI可视化显著提升购买信心和转化率
5. **AI降本增效**：先进AI将造型师工作量减少约25%（来源：Canvas Business Model）

### 5.6 弱点

1. **活跃客户持续下降**：2025财年活跃客户231万，同比下降7.9%，增长乏力
2. **仅限美国市场**：业务集中在美国，未进入中国等亚洲市场
3. **供应链重**：需要大量库存和物流基础设施，运营成本高
4. **等待时间长**：从下单到收货需要数天，无法即时满足
5. **不适应中国市场**：中国消费者习惯即时满足和社交购物，订阅盒子模式接受度低

### 5.7 用户量级

231万活跃客户（FY2025），收入约12.7亿美元（FY2025预估）。

### 5.8 对AiNeed的启示

Stitch Fix是AI造型领域最成功的商业验证，但它的模式不适合中国市场。AiNeed应借鉴其AI推荐算法和Vision可视化理念，但采用"即时推荐+线上购买"而非"订阅盒子"模式。Stitch Fix Vision的75%重复使用率证明：可视化是AI造型的关键转化杠杆。

## 六、美图秀秀/美颜相机 - AI试妆/试衣

### 6.1 产品概述

美图公司（01357.HK）是中国AI影像领域龙头，美图秀秀连续十年蝉联中国图片美化赛道用户规模第一。2025年美图公司经调整净利润9.65亿元，同比增长64.7%。核心产品包括美图秀秀、美颜相机、Wink等，三款产品均入选全球Top 50生成式AI移动应用榜单（来源：美图2025年财报）。

### 6.2 核心功能（AI相关）

| 功能 | 描述 |
|------|------|
| AI换装 | 美颜相机AI换装功能，新增超2,300万全球用户 |
| AI衣橱 | 2025年2月推出，用户上传衣物生成虚拟穿搭 |
| AI合照 | 推动美图秀秀MAU创历史新高，吸引欧洲超300万新增用户 |
| AI Agent | 美图秀秀上线AI Agent，支持一键换装/场景重塑/背景替换 |
| RoboNeo | AI Agent产品，零市场预算MAU突破百万 |
| 美图设计室 | 电商设计AI Agent，自动生成商品套图 |

### 6.3 商业模式

美图的商业模式为**付费订阅+SaaS+广告**。2025年付费订阅用户加速增长，AI Agent功能成为美图设计室收入增长的主要驱动力。海外MAU突破1亿，Vmake（美国市场）年度经常性收入约300万美元。AI功能显著拉动新增和月活（来源：美图2025年财报/东方证券研报）。

### 6.4 优势

1. **用户规模巨大**：美图秀秀连续十年用户规模第一，美颜相机全球MAU超7,200万
2. **AI技术领先**：2025年美图秀秀更新67个功能，AI功能占比91%
3. **AI换装验证成功**：AI换装推动美颜相机在东南亚爆火，单月新增2,300万全球用户
4. **商业化成熟**：付费订阅+SaaS+广告多元变现，净利润9.65亿元
5. **全球化布局**：海外MAU突破1亿，Vmake等海外产品收入增长

### 6.5 弱点

1. **AI换装偏娱乐**：美图的AI换装以"变装/变美"为卖点，偏娱乐化，非实用穿搭推荐
2. **无搭配逻辑**：AI换装是图像合成，不理解搭配规则（色彩/体型/场合）
3. **无衣橱管理**：无个人衣橱数字化，无法基于已有单品推荐
4. **无定制服务**：纯影像工具，不涉及服装定制和购买
5. **无社区**：工具型产品，无穿搭社区和社交互动

### 6.6 用户量级

美图秀秀：MAU创历史新高（未披露具体数字，此前约1.2亿）；美颜相机：全球MAU超7,200万；Wink：MAU超4,000万；海外总MAU超1亿。

### 6.7 对AiNeed的启示

美图是AiNeed在AI换装/试穿领域最直接的竞争威胁。美图的AI换装已验证了用户对"虚拟换装"的需求，但它的产品定位是"变美工具"而非"造型师"。AiNeed的差异化在于：AI换装只是可视化手段，核心价值是"理解用户+推荐搭配"。如果美图未来在AI换装基础上增加搭配推荐功能，将是AiNeed最大的竞争威胁。

## 八、AiNeed的差异化机会

### 8.1 差异化机会一：AI对话式造型师（Agent模式）

**市场空白**：目前没有一款中国App提供真正的"AI对话式造型师"。搭搭的AI推荐是被动标签匹配，小红书靠用户搜索，美图的AI换装是图像合成。AiNeed的GLM-5 Agent模式（LLM+知识图谱+规则引擎）可以实现：

- 用户自然语言描述需求（"明天面试穿什么""梨形身材怎么穿显瘦"）
- AI理解意图+提取参数+调用知识图谱+检索商品+生成搭配方案
- SSE流式输出，体验如同与真人造型师对话

**竞争壁垒**：知识图谱（1,000+条时尚规则）+ 用户画像（体型/色彩季型/风格偏好）+ 商品库的深度整合，不是简单的LLM对话。

### 8.2 差异化机会二：搭配可视化+定制闭环

**市场空白**：搭搭无可视化，美图的AI换装偏娱乐无搭配逻辑，ZEPETO的3D换装是虚拟的不映射真实服装。AiNeed的"文生图穿搭效果图+Q版形象换装+POD定制"组合实现：

- AI推荐搭配 → 文生图生成穿搭效果图（~0.01元/张，成本极低）
- 搭配同步映射到Q版形象（颜色+类型），提供轻量可视化
- 用户喜欢搭配 → 可直接POD定制生产，从推荐到购买一站式闭环

**竞争壁垒**：文生图+Q版形象+POD定制的三合一体验，目前无竞品同时提供。

### 8.3 差异化机会三：设计市集+高端定制双轨定制体系

**市场空白**：搭搭/小红书/得物均无定制服务，ZEPETO只有虚拟定制，Stitch Fix无定制。AiNeed的"POD大众定制+高端私人定制"双轨体系覆盖不同消费层级：

- **POD大众定制**：图案编辑器+EPROLO POD，客单价99-299元，满足个性化需求
- **设计市集**：免费分享设计+仅收定制生产费，AI预审防侵权，形成UGC生态
- **高端私人定制**：工作室入驻+在线沟通+报价+支付+订单跟踪，客单价1,000-10,000元

**竞争壁垒**：从大众定制到高端定制的全价格带覆盖，设计市集的UGC生态形成供给端网络效应。

### 8.4 差异化机会四：Q版形象的情绪价值

**市场空白**：搭搭纯工具无情绪价值，小红书/得物无虚拟形象，ZEPETO的3D形象成本过高。AiNeed的Q版形象（react-native-skia动态绘制）提供：

- 首页Q版形象作为"AI造型师化身"，idle/happy/wave动画增加亲和力
- 换装结果同步映射到Q版形象，提供轻量可视化
- 极低成本（零外包/零GPU/零CDN），MVP极简占位后续迭代POP MART风格

**竞争壁垒**：Q版形象+AI造型师的"人格化"组合，让用户感觉"有一个懂我的造型师朋友"而非"一个冷冰冰的工具"。

## 九、竞争策略建议

### 9.1 防御策略

| 威胁来源 | 防御措施 |
|---------|---------|
| 美图增加AI搭配推荐 | 加速积累用户画像数据和搭配规则，建立数据壁垒；美图擅长图像处理但不擅长时尚知识 |
| 小红书增加AI造型功能 | 定位差异化：小红书是"找灵感"，AiNeed是"找答案"；小红书推荐内容，AiNeed推荐搭配 |
| 搭搭升级AI能力 | AiNeed的对话式AI+可视化+定制闭环是搭搭短期无法复制的组合 |
| 得物扩展定制服务 | 得物聚焦正品交易，定制是不同赛道；AiNeed可与得物CPS合作而非竞争 |

### 9.2 进攻策略

1. **抢占"AI造型师"品类认知**：在用户心智中建立"AiNeed=AI造型师"的品类联想，类似"小红书=种草""得物=正品"
2. **搭搭用户转化**：搭搭200万用户有衣橱管理需求但推荐体验差，AiNeed可提供衣橱导入+升级推荐体验
3. **小红书导流**：在小红书发布AI造型内容（搭配效果图/穿搭日记），引导用户下载AiNeed体验个性化推荐
4. **美颜相机用户交叉**：美颜相机AI换装用户有穿搭需求，AiNeed提供"换装+推荐+购买"完整体验

| 知识图谱 | 1,000+时尚规则+持续扩展，竞品短期无法复制 | 3-6个月 |
