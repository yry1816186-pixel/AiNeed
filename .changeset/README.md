# Changesets

This folder is managed by the `@changesets/cli` package.

## How to use

1. Run `pnpm changeset` to create a new changeset
2. Select the packages that have changed
3. Choose the version bump type (patch/minor/major)
4. Write a description of the change
5. Commit the changeset file along with your code changes
6. Run `pnpm version-packages` to update package versions
