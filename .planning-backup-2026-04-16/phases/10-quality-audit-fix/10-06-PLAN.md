<action>
1. In `PostMasonryCard.tsx`, calculate image height based on aspect ratio:
   ```typescript
   // Instead of: 160 + (idx % 4) * 30
   // Use: Calculate from post.imageAspectRatio or default 4:5
   const IMAGE_BASE_HEIGHT = 200; // base height in dp
   const cardHeight = useMemo(() => {
     const ratio = post.imageAspectRatio ?? 0.8; // default 4:5 portrait
   ```
