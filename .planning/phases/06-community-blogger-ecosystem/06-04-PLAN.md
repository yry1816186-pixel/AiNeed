创建 `apps/mobile/src/screens/CreatePostScreen.tsx` 实现发帖功能（SOC-02, D-19, D-20）：

1. 页面布局:
   - 顶部: 取消/发布按钮
   - 图片网格: 3列网格，最多9张，+号添加
   - 文字输入: 多行 TextInput，placeholder "分享你的穿搭心得..."
   - 标签选择: Chip 多选（OOTD/通勤/约会/运动/休闲/法式/日系/韩系等预设标签）
   - 单品标注: 搜索按钮 → 弹出搜索框 → 从平台商品库选择 → 显示已选单品列表
   - 分类选择: PostCategory 下拉选择

2. 图片上传:
   - 使用 expo-image-picker 选择图片（最多9张）
   - 调用 apiClient.upload() 逐张上传
   - 上传中显示进度条
   - 上传完成显示缩略图，可删除重排

3. 草稿保存（D-19）:
   - 使用 AsyncStorage 存储草稿，key: `draft_post`
   - 每次内容变更 debounce 2秒后自动保存
   - 页面加载时检查草稿，有草稿弹出恢复提示
   - 发布成功后清除草稿

4. 单品标注（D-20）:
   - 搜索框调用 clothing API 搜索商品
   - 选择后添加到 postItems 数组
   - 显示为小卡片（商品图+名称），可删除

5. 提交发帖:
   - 调用 communityApi.createPost({ title, content, images, tags, category, relatedItemIds })
   - 成功后返回社区首页，新帖子出现在列表顶部
