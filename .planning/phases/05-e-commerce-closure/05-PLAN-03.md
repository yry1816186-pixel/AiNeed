# Plan 05-03: Frontend Screen Enhancements

<objective>
Enhance all existing mobile screens for Phase 5 e-commerce features: product detail with SKU selection, cart with coupon support, checkout with real payment, order detail with refund/tracking, and search with multi-dimension filters.
</objective>

<threat_model>
| Threat | Severity | Mitigation |
|--------|----------|------------|
| Payment data interception | High | Use official Alipay/WeChat SDK, never handle raw card data |
| Price tampering on client | High | All price calculations server-side, display-only on client |
| XSS in search input | Medium | Sanitize search input, use TextInput with no HTML rendering |
</threat_model>
