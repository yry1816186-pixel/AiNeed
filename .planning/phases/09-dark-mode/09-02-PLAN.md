## Task 1: App.tsx — StatusBar 动态化 + SplashScreen 主题化

<read_first>
- apps/mobile/App.tsx
- apps/mobile/src/shared/contexts/ThemeContext.tsx
</read_first>

<action>
1. 在 App.tsx 中：
   - 将 `import { theme } from './src/design-system/theme'` 改为仅保留类型导入（如果需要类型），移除对静态 theme 对象的值引用
   - 添加 import: `import { useTheme, createStyles } from './src/shared/contexts/ThemeContext';`

2. 创建一个内部组件 `ThemedAppContent` 来消费 useTheme()（因为 App 本身是 Provider 外层，SplashScreen 部分在 Provider 内部可以消费）:
   ```typescript
   function ThemedStatusBar() {
     const { isDark, colors } = useTheme();
     return (
       <StatusBar
         barStyle={isDark ? "light-content" : "dark-content"}
         backgroundColor={colors.surface}
       />
     );
   }
   ```

3. 替换第203-206行的硬编码 StatusBar:
   ```typescript
   // 之前:
   <StatusBar barStyle="dark-content" backgroundColor={theme.colors.surface} />
   // 之后:
   <ThemedStatusBar />
   ```

4. 替换第255-258行的硬编码 StatusBar:
   ```typescript
   <ThemedStatusBar />
   ```

5. SplashScreen 组件改为接受 colors 参数或使用 useTheme:
   ```typescript
   function SplashScreen() {
     const { colors } = useTheme();
     return (
       <View style={splashStyles(colors).container}>
         <View style={splashStyles(colors).logoContainer}>
           <Ionicons name="shirt-outline" size={48} color={colors.surface} />
         </View>
         <Text style={splashStyles(colors).brandName}>寻裳</Text>
         <ActivityIndicator size="large" color={colors.primary} style={styles.loader} />
       </View>
     );
   }
   ```

6. 将 StyleSheet.create 中的静态 theme 引用改为 createStyles 模式:
   ```typescript
   const splashStyles = createStyles((colors) => ({
     container: {
       flex: 1,
       backgroundColor: colors.surface,
       alignItems: 'center',
       justifyContent: 'center',
     },
     logoContainer: {
       width: 96,
       height: 96,
       borderRadius: 48,
       backgroundColor: colors.primary,
       alignItems: 'center',
       justifyContent: 'center',
       marginBottom: 16,
     },
     brandName: {
       fontSize: 28,
       fontWeight: '700',
       color: colors.primary,
       letterSpacing: 1.5,
       marginBottom: 32,
     },
   }));
   ```

7. 移除 App.tsx 中所有对 `theme.colors.xxx` 的直接引用
</action>

<acceptance_criteria>
- `App.tsx` 不包含 `barStyle="dark-content"` 硬编码
- `App.tsx` 不包含 `barStyle="light-content"` 硬编码
- `App.tsx` 包含 `ThemedStatusBar` 组件定义
- `App.tsx` 包含 `useTheme` 导入
- `App.tsx` 中 `theme.colors` 引用数量为 0（`grep -c "theme\.colors" App.tsx` = 0）
- StatusBar barStyle 根据 isDark 动态切换
- TypeScript 编译通过
</acceptance_criteria>
