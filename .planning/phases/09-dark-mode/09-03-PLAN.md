## Task 1: AI Stylist 屏幕迁移

<read_first>
- apps/mobile/src/features/stylist/screens/AiStylistScreen.tsx
- apps/mobile/src/features/stylist/screens/AiStylistChatScreen.tsx
- apps/mobile/src/features/stylist/screens/OutfitDetailScreen.tsx
- apps/mobile/src/features/stylist/screens/SessionCalendarScreen.tsx
- apps/mobile/src/shared/contexts/ThemeContext.tsx
</read_first>

<action>
1. 对每个 Stylist 屏幕执行标准迁移:
   - 替换 `import { theme } from '../../../design-system/theme'` 为 `import { useTheme, createStyles } from '../../../shared/contexts/ThemeContext'`
   - 添加 `const { colors, seasonAccent } = useTheme();`（AiStylistChatScreen 已使用 seasonAccent，需扩展）
   - 替换所有 `theme.colors.xxx` → `colors.xxx`
   - 替换 StyleSheet.create → createStyles
   - 替换硬编码 `backgroundColor: "#FFFFFF"` → `colors.surface`

2. AiStylistChatScreen 特别注意:
   - 已有两处 useTheme() 调用获取 seasonAccent，合并为一处: `const { colors, seasonAccent } = useTheme();`
   - 聊天气泡颜色需适配深色模式
</action>

<acceptance_criteria>
- `grep -l "import.*theme.*from.*design-system/theme" apps/mobile/src/features/stylist/screens/*.tsx` 返回空
- 每个文件包含 `useTheme()` 调用
- `grep -c '"#FFFFFF"' apps/mobile/src/features/stylist/screens/AiStylistChatScreen.tsx` = 0
- TypeScript 编译通过
</acceptance_criteria>
