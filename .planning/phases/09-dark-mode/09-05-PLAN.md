<action>

   ```typescript

     elevated: "#1F1B18",
     elevated2: "#252019",
     elevated3: "#2A251E",
     elevated4: "#302A23",
     elevated5: "#362F28",
   }
   ```

2. 在 `PaperThemeProvider.tsx` 中修改深色主题 elevation 配置:
   ```typescript
   // Dark theme elevation — 使用不同层级颜色
   elevation: {
     level0: "transparent",
     level1: c.backgrounds.elevated,   // #1F1B18
     level2: c.backgrounds.elevated2,  // #252019
     level3: c.backgrounds.elevated3,  // #2A251E
     level4: c.backgrounds.elevated4,  // #302A23
     level5: c.backgrounds.elevated5,  // #362F28
   }
   ```

3. 审计所有使用 `elevation:` 数字值的文件（约92处），对关键组件添加暗色模式阴影支持:
   - 在深色模式下，为使用 elevation 的组件添加:
     ```typescript
     shadowColor: colors.textPrimary,
     shadowOffset: { width: 0, height: 2 },
     shadowOpacity: isDark ? 0.3 : 0.1,
     shadowRadius: 4,
     ```
   - 优先处理用户可见的浮动元素: FAB、浮动卡片、弹窗
