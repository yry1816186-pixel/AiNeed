# Plan 03: 排版 Token 迁移 Codemod

## Objective

将所有硬编码 fontSize（~856 处）、fontWeight（~551 处）、lineHeight（~132 处）迁移至 DesignTokens.typography.* 语义化 Token。

## Tasks

### Task 03-02: 创建排版迁移 Codemod 并执行

<objective>
编写 ts-morph codemod 脚本，将硬编码排版值替换为 DesignTokens.typography 引用，然后执行迁移。
</objective>

<read_first>
- apps/mobile/scripts/semantic-typography-map.json
- apps/mobile/src/theme/tokens/typography.ts
- apps/mobile/src/theme/tokens/design-tokens.ts
</read_first>

<action>
1. 创建 `apps/mobile/scripts/codemod-typography.ts`：

```typescript
// 使用 ts-morph 解析 TSX/TS 文件
// 逻辑:
// 1. 加载 semantic-typography-map.json
// 2. 遍历 src/ 下所有 .tsx/.ts 文件（排除 node_modules, theme/tokens/）
// 3. 在 StyleSheet.create() 和内联样式中查找排版属性
// 4. 替换规则:
//    - fontSize: <number> → DesignTokens.typography.styles.xxx.fontSize
//    - fontWeight: '<number>' → DesignTokens.typography.fontWeights.xxx
//    - lineHeight: <number> → DesignTokens.typography.styles.xxx.lineHeight
// 5. 优化: 如果同一对象中 fontSize + fontWeight + lineHeight 可组合为完整 TextStyle，
//    则替换为 DesignTokens.typography.styles.xxx（展开属性）
// 6. 添加 import { DesignTokens } from '...' 如果不存在
// 7. 保留原始值作为注释: /* was: 14 */
// 8. 支持 --dry-run 和 --file <path> 模式
```

2. 在 `apps/mobile/package.json` 中添加脚本：

```json
{
  "scripts": {
    "codemod:typography": "npx ts-node scripts/codemod-typography.ts",
    "codemod:typography:dry-run": "npx ts-node scripts/codemod-typography.ts --dry-run"
  }
}
```

3. 执行迁移：
   - 先 dry-run：`cd apps/mobile && npx ts-node scripts/codemod-typography.ts --dry-run`
   - 审查报告
   - 执行：`cd apps/mobile && npx ts-node scripts/codemod-typography.ts`
   - 类型检查：`cd apps/mobile && npx tsc --noEmit`

4. 手动处理无法自动映射的值（非标准 fontSize 值等）
</action>

<acceptance_criteria>
- `apps/mobile/scripts/codemod-typography.ts` 文件存在
- `apps/mobile/package.json` 包含 `codemod:typography` 脚本
- `cd apps/mobile && npx tsc --noEmit` 无新增类型错误
- `cd apps/mobile && npx ts-node scripts/audit-hardcoded-typography.ts` 报告用户可见组件中硬编码 fontSize = 0
- `cd apps/mobile && npx ts-node scripts/audit-hardcoded-typography.ts` 报告用户可见组件中硬编码 fontWeight = 0
- 原始值保留为注释 `/* was: 14 */`
</acceptance_criteria>
