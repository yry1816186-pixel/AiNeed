# Plan 01: 迁移 identity 域模块

## Objective

将 auth, users, profile, onboarding, privacy 五个模块从 `src/modules/` 迁移到 `src/domains/identity/`，更新所有引用路径，确保编译和测试通过。

## Threat Model

| Threat ID | Description | Severity | Mitigation |
|-----------|-------------|----------|------------|
| T-3-01 | PII 加密代码在迁移中被意外修改或遗漏 | High | SecurityModule 保持 @Global 不迁移；PII 相关文件标记不可移动；迁移后运行 PII 加密测试验证 |
