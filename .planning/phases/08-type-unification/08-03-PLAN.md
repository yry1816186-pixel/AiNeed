# Plan 08-03: ApiResponse/PaginatedResponse + BodyAnalysis 统一

<objective>
统一 `ApiResponse` 和 `PaginatedResponse` 定义（合并 meta + message 字段），统一 `BodyAnalysis` 为单一 camelCase 定义。
</objective>

<threat_model>
**Threat:** ApiResponse 添加 message 字段可能影响后端序列化逻辑
**Mitigation:** 后端未引用 @xuno/types，不影响运行时；message 字段为 optional，不破坏现有消费者
**Severity:** Low
</threat_model>

<tasks>

### Task 8-03-01: 统一 ApiResponse 和 PaginatedResponse

<read_first>
- packages/types/src/api.ts (08-01 创建)
- apps/mobile/src/types/api.ts
- packages/shared/src/types/api.ts
</read_first>

<action>
1. 在 `packages/types/src/api.ts` 中定义统一的 `ApiResponse`：

```typescript
export interface ApiError {
  code: string;
  message: string;
  details?: Record<string, unknown>;
}

export interface PaginationMeta {
  page?: number;
  limit?: number;
  total?: number;
  totalPages?: number;
}

export interface ApiResponse<T = unknown> {
  success: boolean;
  data?: T;
  error?: ApiError;
  message?: string;
  meta?: PaginationMeta;
}

export interface PaginatedResponse<T> {
  items: T[];
  page: number;
  pageSize: number;
  limit: number;
  total: number;
  totalPages: number;
  hasMore?: boolean;
}

export interface PaginationParams {
  page?: number;
  pageSize?: number;
  sortBy?: string;
  sortOrder?: 'asc' | 'desc';
}
```

关键决策：
- `ApiResponse` 同时包含 `message?: string`（mobile 需要）和 `meta?: PaginationMeta`（共享包原有）
- `PaginatedResponse` 同时包含 `pageSize` 和 `limit`（两者都保留，`limit` 标记为 `@deprecated`）
- `PaginationMeta` 新增 `totalPages` 字段（@xuno/shared 有此字段）
- `ApiError` 保持与现有定义一致

2. 更新 `apps/mobile/src/types/api.ts`：
   - 删除本地 `ApiResponse`, `ApiError`, `PaginatedResponse`, `PaginationParams` 定义
   - 从 `@xuno/types` 导入
   - 保留 mobile 特有类型：`ClothingAnalysisResult`, `SimilarItemResult`, `OutfitRecommendationResult`, `VirtualTryOnResult`, `SearchFilters`, `CartItem`, `Order`, `OrderStatus`, `Address`
</action>

<acceptance_criteria>
- `ApiResponse` 同时有 `message?: string` 和 `meta?: PaginationMeta`
- `PaginatedResponse` 有 `pageSize` + `limit`
- mobile `api.ts` 中 `ApiResponse`, `ApiError`, `PaginatedResponse`, `PaginationParams` 从 @xuno/types 导入

- `cd apps/mobile && npx tsc --noEmit` 通过
