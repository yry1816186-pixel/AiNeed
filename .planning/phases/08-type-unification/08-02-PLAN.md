# Plan 08-02: ClothingItem 拆分 + ClothingCategory 统一

<objective>
将 `ClothingItem` 拆分为 `CatalogItem`（商品目录项）和 `WardrobeItem`（衣柜单品），统一 `ClothingCategory` 为 mobile 扩展版（12 值），消除语义分裂。
</objective>

<threat_model>
**Threat:** ClothingItem 重命名为 CatalogItem 可能破坏后端 API 响应类型
**Mitigation:** 后端目前未引用 @xuno/types，不影响 API 契约；mobile 端逐步迁移引用
**Severity:** Low
</threat_model>

<tasks>
