# Plan 08-01: 核心类型统一 — enum→type alias + 日期统一 + User 扩展

<objective>
将 `@xuno/types` 中所有 `enum` 转换为 `type` alias，统一日期字段为 `string`，扩展 `User` 类型以包含 mobile 需要的字段，消除 mobile 中 `SharedXxx` 别名 workaround。
</objective>

<threat_model>
**Threat:** enum→type alias 转换可能破坏使用 `Gender.Female` 等枚举成员访问的代码
**Mitigation:** 全局搜索 `Gender.`, `BodyType.`, `SkinTone.` 等枚举成员引用，逐一替换为字面量值；保留 `const` 对象作为运行时映射（如需要）
**Severity:** Medium
</threat_model>

<tasks>

5. 创建 `packages/types/src/wardrobe.ts` — WardrobeItem, ClothingStyle, Season, Occasion

9. 创建 `packages/types/src/api.ts` — ApiResponse, ApiError, PaginatedResponse, PaginationParams
10. 创建 `packages/types/src/common.ts` — Nullable, Optional, Maybe, DeepPartial, PickRequired, OmitKeys, ValueOf, KeysOf, ArrayElement
11. 更新 `packages/types/src/index.ts` — 仅做 re-export

每个文件中的 enum 全部替换为 type alias。例如：
```typescript
// Before

  Male = 'male',
  Female = 'female',
  Other = 'other',
}

// After
export type Gender = 'male' | 'female' | 'other';
```
