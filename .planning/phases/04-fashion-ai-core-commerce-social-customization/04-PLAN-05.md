## Task 05: 安装并配置 eslint-plugin-boundaries

<read_first>
- apps/backend/.eslintrc.json
- apps/backend/package.json
</read_first>

<action>
1. 安装依赖：
   ```bash
   cd apps/backend && pnpm add -D eslint-plugin-boundaries
   ```

2. 更新 `apps/backend/.eslintrc.json`，添加 boundaries 配置：
   ```json
   {
     "plugins": ["boundaries"],
     "rules": {
       "boundaries/element-types": [2, {
         "default": "allow",
         "rules": [
           {
             "from": ["fashion"],
             "allow": ["identity", "platform", "common", "fashion"]
           },
           {
             "from": ["ai-core"],
             "allow": ["identity", "platform", "common", "ai-core"]
           },
           {
             "from": ["commerce"],
             "allow": ["identity", "fashion", "platform", "common", "commerce"]
           },
           {
             "from": ["social"],
             "allow": ["identity", "fashion", "platform", "common", "social"]
           },
           {
             "from": ["customization"],
             "allow": ["identity", "fashion", "platform", "common", "customization"]
           },
           {
             "from": ["identity"],
             "allow": ["platform", "common", "identity"]
           },
           {
             "from": ["platform"],
             "allow": ["common", "platform"]
           }
         ]
       }],
       "boundaries/no-internal": [2, {
         "default": "disallow",
         "rules": [
           {
             "from": ["fashion", "ai-core", "commerce", "social", "customization", "identity", "platform"],
             "allow": ["*"]
           }
         ]
       }]
     },
     "overrides": [{
       "files": ["src/domains/**/*.ts"],
       "rules": {
         "boundaries/element-types": 2
       }
     }],
     "settings": {
       "boundaries": {
         "elements": [
           {
             "type": "fashion",
             "pattern": "src/domains/fashion/**/*"
           },
           {
             "type": "ai-core",
             "pattern": "src/domains/ai-core/**/*"
           },
           {
             "type": "commerce",
             "pattern": "src/domains/commerce/**/*"
           },
           {
             "type": "social",
             "pattern": "src/domains/social/**/*"
           },
           {
             "type": "customization",
             "pattern": "src/domains/customization/**/*"
           },
           {
             "type": "identity",
             "pattern": "src/domains/identity/**/*"
           },
           {
             "type": "platform",
             "pattern": "src/domains/platform/**/*"
           },
           {
             "type": "common",
             "pattern": "src/common/**/*"
           }
         ]
       }
     }
   }
   ```

3. 运行 `cd apps/backend && npx eslint src/domains/ --no-error-on-unmatched-pattern` 验证规则生效
</action>
