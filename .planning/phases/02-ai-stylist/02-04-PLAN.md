重构 `AiStylistScreen.tsx`（替换现有内容），实现方案页交互模式：

**页面布局（从上到下）：**
1. **顶部导航栏**：标题"AI 造型师" + 历史按钮（日历图标）
2. **方案展示区**（主内容区）：
   - 无方案时：空状态（图标 + "告诉 AI 你想穿什么" + 预设问题引导）
   - 有方案时：`OutfitPlanView` 组件
3. **输入区**（底部固定）：
   - `SceneQuickButtons` 组件（6 个场景按钮）
   - 文字输入框 + 发送按钮

**交互流程：**
1. 首次进入：调用 `fetchPresetQuestions()`，如果是新用户显示 `PresetQuestionsModal`
2. 点击场景按钮或输入文字：调用 `createSession()` + `sendMessage()`
3. 收到方案后：调用 `fetchOutfitPlan()`，显示 `OutfitPlanView`
4. 点击单品"替换"：调用 `fetchAlternatives()`，显示 `ItemReplacementModal`
5. 选择替换商品：调用 `replaceItem()`，刷新方案
6. 点击反馈按钮：显示 `FeedbackModal`，提交后调用 `submitFeedback()`
7. 点击历史按钮：导航到 `SessionCalendarScreen`

**关键实现：**
- 使用 `useAiStylistStore` 替代所有 `useState`
- 获取用户位置（从 `homeStore` 的天气数据或 `navigator.geolocation`）传递给 `sendMessage`
- 方案生成中显示加载动画（脉冲点 + "AI 正在为你搭配..."）
- 错误状态显示重试按钮

**组件导入：**
```typescript

  OutfitPlanView,
```
