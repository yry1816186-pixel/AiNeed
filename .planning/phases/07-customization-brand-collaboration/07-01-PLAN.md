# Plan 07-01: Customization Editor Backend + Template System + Pricing Engine

## Objective

Build the backend infrastructure for the 2D customization editor: product templates, design data persistence, automatic pricing engine, and AI preview generation endpoint. Extend existing CustomizationService to support editor-based customization flow.

## Context

@07-CONTEXT.md -- decisions D-01 through D-10, D-06
@apps/backend/src/modules/customization/customization.service.ts -- existing CRUD
@apps/backend/prisma/schema.prisma -- CustomizationRequest/Quote models

## Threat Model

| File | Surface | Mitigation |
|------|---------|------------|
| pricing-engine.ts | pricing calculation | validate all inputs, no negative prices |
| customization.controller.ts | file upload | limit file size, validate MIME types |

### Task 3: Build Automatic Pricing Engine

<task type="auto">
<read_first>
- apps/backend/src/modules/customization/customization.service.ts
- apps/backend/src/modules/customization/templates/customization-templates.ts
</read_first>
<action>
1. Create `apps/backend/src/modules/customization/pricing/pricing-engine.ts`:
   - `PricingEngine` class with method `calculatePrice(params: PricingParams): PricingResult`
   - PricingParams: templateType, designComplexity (layer count), hasTextLayers, imageCount, printSide ('front' | 'back' | 'both')
   - PricingResult: basePrice, complexitySurcharge, textSurcharge, sideSurcharge, totalPrice, currency, estimatedDays
   - Formula: totalPrice = basePrice + (layerCount > 5 ? layerCount * 5 : 0) + (hasText ? 10 : 0) + (printSide === 'both' ? basePrice * 0.3 : 0)
   - Estimated days: base 3 + (layerCount > 10 ? 2 : 0) days
   - Validate all inputs, no negative prices
   - Export PricingEngine instance

2. Create `apps/backend/src/modules/customization/pricing/index.ts` barrel export
</action>
<acceptance_criteria>
- PricingEngine.calculatePrice returns valid PricingResult for all template types
- No negative prices possible
- Complexity surcharge kicks in at >5 layers
- Text layer adds 10 CNY
- Both-sides printing adds 30% of base price
- Estimated days calculated correctly
</acceptance_criteria>
</task>
