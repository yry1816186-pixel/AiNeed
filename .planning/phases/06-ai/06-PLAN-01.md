### Task 04: 更新 Dockerfile

<type>modify</type>

</files>

<action>
更新 Dockerfile 以支持 pyproject.toml：

1. 将 `COPY requirements.txt .` 改为 `COPY pyproject.toml requirements.txt ./`
2. 将 `RUN pip install --no-cache-dir --prefix=/install -r requirements.txt` 改为：
   ```
   RUN pip install --no-cache-dir --prefix=/install -r requirements.txt && \
       pip install --no-cache-dir --prefix=/install -e /build
   ```
   注意：由于 Dockerfile 的 WORKDIR 是 /build，需要先复制整个项目再安装。
   
   更好的方案：使用两阶段构建，先复制 pyproject.toml 安装依赖，再复制源码安装包：
   ```
   # Builder stage
   FROM python:3.11-slim AS builder
   WORKDIR /build
   RUN apt-get update && \
       apt-get install -y --no-install-recommends build-essential gcc && \
       rm -rf /var/lib/apt/lists/*
   COPY requirements.txt .
   RUN pip install --no-cache-dir --prefix=/install -r requirements.txt
   COPY . .
   RUN pip install --no-cache-dir --prefix=/install -e .

   # Runtime stage
   FROM python:3.11-slim AS runtime
   RUN apt-get update && \
       apt-get install -y --no-install-recommends libgl1-mesa-glx libglib2.0-0 curl && \
       rm -rf /var/lib/apt/lists/*
   RUN groupadd -r xuno && useradd -r -g xuno -d /app -s /sbin/nologin xuno
   COPY --from=builder /install /usr/local
   WORKDIR /app
   COPY . .
   ENV PYTHONDONTWRITEBYTECODE=1 \
       PYTHONUNBUFFERED=1 \
       PYTHONPATH=/app \
       ENVIRONMENT=production \
       PORT=8002
   RUN mkdir -p /data/images /tmp/xuno && \
       chown -R xuno:xuno /app /data /tmp/xuno
   USER xuno
   EXPOSE 8002
   HEALTHCHECK --interval=30s --timeout=10s --start-period=40s --retries=3 \
       CMD curl -f http://localhost:8002/health || exit 1
   CMD ["uvicorn", "ml.api.main:app", "--host", "0.0.0.0", "--port", "8002", "--workers", "1", "--log-level", "info"]
   ```

3. 将 CMD 中的 `api.main:app` 改为 `ml.api.main:app`（因为 PYTHONPATH=/app，包从 /app 根目录解析）
</action>
